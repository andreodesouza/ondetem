Contexto

O Problema que Resolvemos

O mercado de beleza e estetica enfrenta desafios de digitalizacao e conexao entre profissionais e clientes.

Dificuldade de Encontrar

Agendamento Manual

Interface

Tela Principal - Estilo Airbnb

Layout split-screen com listagem rolavel a esquerda e mapa interativo a direita.

Split Screen Layout

Inspirado no design do Airbnb: lado esquerdo com cards de saloes rolaveis e lado direito com mapa fixo mostrando localizacoes.

Busca Inteligente

Barra de busca com filtragem em tempo real por nome do salao ou tipo de servico, com versao desktop e mobile.

Filtro por Categorias

Botoes de categoria (Cabelo, Manicure, Depilacao, Sobrancelhas, Massagem) com icones e feedback visual.

Mapa Interativo (Leaflet)

Marcadores coloridos para cada salao. Ao clicar no card, o mapa centraliza no estabelecimento com animacao flyTo. Calculo de distancia via Haversine.

Responsividade Mobile

No mobile, layout se transforma em gaveta deslizante (drawer) sobre o mapa, com barrinha de arrastar e categorias sticky.

Geolocalizacao

Deteccao automatica da posicao do usuario via GPS, com calculo de distancia para cada estabelecimento.

11 / 15
PWA

Progressive
Web App

Aplicacao instalavel no dispositivo com capacidades nativas, funcionando offline e com notificacoes push.

Instalavel

Manifest.json configurado para instalacao como app nativo, com icones, splash screen e orientacao portrait.

Funciona Offline

Service Worker com cache de arquivos criticos (HTML, CSS, JS, Bootstrap) usando estrategia Cache-First.

Push Notifications

Sistema VAPID para notificacoes push. Lembretes automaticos no dia e 1 hora antes dos agendamentos.

// service-worker.js

const CACHE_NAME = 'belezaja-cache-v1';
const urlsToCache = [
  './',
  './index.html',
  './style.css',
  './script.js',
  'bootstrap.min.css'
];

self.addEventListener('install', event => {
  event.waitUntil(
    caches.open(CACHE_NAME)
      .then(cache => cache.addAll(urlsToCache))
  );
});

self.addEventListener('push', event => {
  const dados = event.data?.json();
  self.registration.showNotification(
    dados.title, { body: dados.body }
  );
});

Seguranca

Seguranca e Validacao

Mecanismos de protecao e validacao em todo o sistema.

Validacao de Senha

Indicador visual de forca da senha com 4 niveis (Muito fraca, Fraca, Boa, Forte). Verifica comprimento, maiusculas, numeros e caracteres especiais.

Algoritmo de Luhn

Validacao matematica do numero do cartao de credito, garantindo que apenas numeros validos sao aceitos antes de processar.

Controle de Sessao

Sistema de autenticacao com localStorage, redirecionamento baseado em tipo de usuario e guard de rotas nos paineis.

Mascaras e Formatacao

Formatacao automatica de campos (cartao, validade, CVV, telefone). Prevencao de entrada de dados invalidos com sanitizacao em tempo real.

13 / 15
Recursos Adicionais

Funcionalidades Extras

Detalhes que enriquecem a experiencia do usuario e diferenciam a plataforma.

Sistema de Favoritos

Clientes podem favoritar saloes com toggle visual (coracao). Favoritos persistem no localStorage e sao exibidos no painel pessoal.

Lembretes Inteligentes

Notificacoes automaticas no dia do agendamento e 1 hora antes. Sistema de limpeza automatica de notificacoes antigas (+2 dias).

Geocodificacao Reversa

Empresas buscam endereco pelo nome da rua via Nominatim API. Resultado posiciona marcador no mapa, arrastavel para ajuste fino.

Gestao de Agenda

Empresas configuram slots de horario por dia da semana. Interface visual com badges removiveis para cada slot de atendimento.

Filtros Avancados

Filtros de agendamentos por status (Confirmado, Pendente, Concluido, Cancelado) com dropdown elegante no painel do cliente.

Gestao de Conta

Edicao de perfil, alteracao de senha com indicador de forca, e exclusao de conta com modal de confirmacao e remocao total de dados.
